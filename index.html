Seattle Resident Service Requests

Data Analysis and Visualizations

Project Overview

This project explores customer service request data collected by the City of Seattle. Using data visualizations, we reveal trends across time, neighborhoods, service types, and departments, while creating a metric to help prioritize neighborhood support.

Key Questions

  • What are the trends by month, quarter, day of week, hour, and year?
  • Which neighborhoods and departments have the highest service requests?
  • What service types are growing or declining?
  • How can neighborhoods be ranked based on service performance?

Neighborhood Metric

We developed a custom ranking formula to assess the performance of each neighborhood based on request volume, request types, and changes over time. Neighborhoods are prioritized for government attention and resources accordingly.
